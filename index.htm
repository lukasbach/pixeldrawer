About pixel drawer

Hi, my name is Lukas Bach, and i am a german web developer. Pixel drawer is just a small project and attempt, to create an web-app. It is obviously not an actual drawing software, and the main reason I created it, was just because it was fun to do so.

Pixel drawer is mainly based on HTML, CSS and lots and lots of JavaScript. I haven't been using JavaScript that much before, so I trained myself in that as this is probably one of the first projects of mine with such an excessive use of JavaScript. It also uses some of the newer features introduces with HTML5, such as the Web Storage, where the JavaScript saves the images on the Computer.

Only for publishing and for downloading images, the image data (an JavaScript object, which contains meta information to the drawing as well as an array for the actual image) is transferred to the server and processed by PHP (converted to JSON before transfer). I have been using PHP often before, so I am pretty much used to it, although I only used it in the procedural way most of the time.

Feel free to try to create some images, you can submit them to the public gallery if you want. I hope you have fun!
